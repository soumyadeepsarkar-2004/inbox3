## 🎯 What Is Inbox3?

Inbox3 is a **fully-decentralized messaging platform** built on Aptos blockchain:

- 💬 **Direct Messages** — Encrypted 1-on-1 chats
- 👥 **Group Chat** — Unlimited members
- 📁 **File Sharing** — IPFS storage
- 🎥 **Voice/Video** — P2P calls
- 🔒 **Encrypted** — End-to-end security
- ⚡ **Fast** — Blockchain powered
- 🌍 **Decentralized** — No central server

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 19 + TypeScript + Tailwind CSS 4 |
| Blockchain | Aptos (Move language) |
| Storage | IPFS (via Pinata) |

## 🆘 Quick Help

### I can't connect my wallet
**Solution**: Install [Petra wallet](https://petra.app/) browser extension

### I don't see messages saving
**Solution**: Check Pinata API keys are correct in Vercel environment variables

### The app is slow
**Solution**: Check your internet speed, or clear browser cache

### I want to call someone
**Solution**: Make sure signaling server is configured (optional feature)
